Game Details

Performance Considerations

ELEVEN 28 is optimized to run smoothly on a range of hardware:

  • Low-end systems: The game supports reduced graphical settings and simplified UI options for better performance.
  • Mid-range setups: Players can expect a stable 60 FPS at 1080p with recommended settings.
  • High-end systems: The game fully utilizes multi-core processors and high-performance GPUs for 4K visuals, ultra-smooth animations, and real-time physics.

Storage & Installation

  • Initial Download Size: ~20GB
  • Post-Updates & Expansions: May require up to 50GB of free space
  • SSD Recommended: For faster loading times and smoother simulation transitions

Online & Multiplayer Requirements

  • Single-player Mode: Can be played offline, but online updates are recommended for live database updates and patches
  • Multiplayer Features: Online leagues and competitive modes require a stable broadband connection (10 Mbps recommended)

Compatibility & Additional Software

  • Supported Resolutions: 1080p, 1440p, 4K (Ultrawide monitor support included)
  • Controller Support: Full compatibility with Xbox and PlayStation controllers

Required Software:

  • DirectX 12 (Windows)
  • Vulkan API (Linux)
  • .NET Framework 4.8
  • Anti-cheat system (for online play)

Accessibility Features

ELEVEN 28 includes various accessibility options:

  • Colorblind Modes: Protanopia, Deuteranopia, and Tritanopia-friendly UI
  • UI Scaling: Adjustable font sizes and HUD elements
  • Input Customization: Key remapping and full controller customization
  • Subtitles & Audio Descriptions: Support for match commentary and in-game menus
